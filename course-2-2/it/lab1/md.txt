Какие задачи позволяет решать надстройка Поиск решения?
Оптимизация производственных процессов, планирование перевозок, составление финансовых планов и др.

Какая функция называется целевой?
Функция, значение которой необходимо минимизировать или максимизировать.

Зачем служит элемент Равно в диалоговом окне Поиск решения?
Для задания ограничения на целевую функцию.

Какой метод поиска был применен для решения задачи оптимизации?
Метод линейного программирования.

Зачем служит элемент Предположить диалогового окна Поиск решения?
Для предварительной установки значений ячеек.

Какие типы отчетов возможны в результате найденного решения?
Отчеты "Результат", "Устойчивость", "Пределы".
